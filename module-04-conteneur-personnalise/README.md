# Module 4 : Conteneur personnalisé

## Objectifs pédagogiques

- Maîtriser la création de Dockerfiles
- Comprendre les instructions Dockerfile et leurs usages
- Appliquer les bonnes pratiques de construction d'images
- Optimiser les images avec multi-stage builds
- Créer des images pour différents langages et frameworks

## Prérequis

- Modules 1, 2 et 3 complétés
- Compréhension des images et layers
- Connaissance de base en développement

## TPs disponibles

1. **TP8 - Dockerfile fondamentaux** : Créer vos premières images
2. **TP9 - Optimisation et bonnes pratiques** : Images production-ready
3. **TP10 - Multi-stage builds** : Images optimisées et sécurisées

## Durée estimée

3 heures
