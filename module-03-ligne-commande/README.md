# Module 3 : Ligne de commande avancée

## Objectifs pédagogiques

- Maîtriser la gestion des volumes Docker pour la persistance des données
- Comprendre et configurer les réseaux Docker
- Gérer efficacement les images (création, tags, push/pull)
- Nettoyer et optimiser son environnement Docker

## Prérequis

- Modules 1 et 2 complétés
- Docker installé et fonctionnel
- Compréhension des commandes de base Docker

## TPs disponibles

1. **TP5 - Volumes et persistance** : Gérer les données persistantes
2. **TP6 - Réseaux Docker** : Créer et configurer des réseaux
3. **TP7 - Gestion avancée des images** : Tags, registres, optimisation

## Durée estimée

2 heures
