# Module 1 : De la virtualisation à Docker

## Objectifs pédagogiques

- Comprendre les différences entre virtualisation et conteneurisation
- Installer Docker sur différents systèmes d'exploitation
- Vérifier l'installation et exécuter son premier conteneur

## Prérequis

- Connaissances de base en ligne de commande
- Droits d'administration sur votre machine
- Connexion Internet

## TPs disponibles

1. **TP1 - Comparaison VM vs Conteneur** : Comprendre les concepts fondamentaux
2. **TP2 - Installation de Docker** : Installer Docker sur votre système
